# 第121章　神的力量

天淵谷的空氣凝固了。

不是比喻，是字面意義上的凝固——那一刻，所有人的肺都停止了擴張，風停了，雲停了，連草葉上的露水都忘記了滴落，整個世界像是被一隻無形的手捏住，強行按下了靜止鍵。

然後，那隻手輕輕一抬。

沒有法術，沒有咒語，甚至沒有任何可見的能量波動。只是一個普通老人的普通動作——就像一個坐了太久的老頭站起來活動手腳。

重力翻轉了。

距離谷心三百步的區域內，大地不再向下，而是向上。八名契士來不及反應，腳下的土地從他們的鞋底撕裂，整個人向天空「墜落」。他們驚叫著試圖展開靈契，但靈力在離體的瞬間就像被一隻看不見的手掐住，嗤的一聲，散了。

八個人在半空中失去了所有靈力。

然後重力回歸正常，他們真正向下墜落。

蘇長淵揮袖，氣流托住了其中四人；赫連峰大步衝出，硬生生用血肉之軀接住了另外兩人；最後兩人落地，骨折，昏迷，沒有死，但已失去戰力。

這一切發生在五秒之內。

契神站在天淵谷的正中心，他的外表依然是那個普通的老人——白髮，皺紋，一件洗得發白的舊袍子，如果放在街頭，絕對是那種被年輕人讓座的老大爺形象。但此刻他站在那裡，就像一塊被插入棋盤正中心的異物，整個天地都在他周圍悄悄變形，宛如一張薄紙被頂著它中心的手指微微拱起。

他低頭看了看自己的手，像是在確認什麼。

「千年了。」他的聲音很輕，像是在自言自語，「真的千年了。」

沒有人敢說話。

天衍大陸有史以來最強的陣仗——三大家族的契聖坐鎮，十七名契王環繞，一百三十餘名契師精英，外加各家族的壓箱底底牌。這個陣容放在任何時代，都足以橫掃一切已知威脅。

但所有人都感受到了同一種東西：壓迫。

不是力量的壓迫，不是靈力的壓迫，而是某種更根本的東西——就像一隻螞蟻感受到人腳踏下的那種，不是傷害，是存在本身的不對等。

蘇念卿站在中陣，霜翎趴在她肩頭，銀色的毛髮豎了起來。那十五條尾巴微微顫抖，不是寒意，是某種本能的恐懼。靈狐的感知比人類更敏銳，霜翎感受到的東西蘇念卿猜不到，但那種顫抖讓她握緊了拳頭。

她掃視四周，所有人——她的父親，赫連峰，慕容玄，所有她認識的強者——此刻都有一個共同的表情：茫然。

那是從未遭遇過的茫然。他們練了一輩子靈契，戰了一輩子，但從來沒有遇到過這種感覺——對面的存在讓他們的所有經驗全部失效，就像一個只玩過棋盤遊戲的人突然被放進了現實戰場，規則不一樣了，完全不一樣了。

然後她看到了白無極。

白無極站在所有人的最前面。

他是唯一一個站得筆直的人。

周圍所有人都在那股無形的壓力下弓著背，有人跪了半膝，有人扶著地，就連蘇長淵那樣的契聖都在用靈力硬撐著維持站姿——但白無極站在那裡，像一根釘進地裡的木樁，既沒有靈力波動，也沒有任何防禦姿態，就是站著，雙臂自然垂落，甚至連表情都沒有特別凝重，只是皺著眉頭，帶著一種「這是什麼情況」的正常困惑。

黃金體驗·Requiem在他的外圍無聲運轉。

因果歸零。

那股針對所有人的壓力，對他而言不是壓力，是一陣風，是一片落葉，是完全無法抵達的東西。它不是被擋住的，它是在抵達之前就被抹除了「會傷害白無極」這個可能性本身。

契神的目光落在他身上，停了一秒。

那是第一次，契神的目光有了焦點。

白無極感覺到那道視線，對視了回去。在那雙看起來渾濁的老人眼睛裡，他看到了某種東西——不是人類的那種情緒，而是某種更古老、更平靜的東西，像是深淵在看他。

「有意思。」契神說。

他又抬了抬手，這次力道稍稍重了一些。

方圓兩百步，大地龜裂，向上翻卷，石塊向四面八方飛射，不是爆炸，是像有人用手指輕輕撥了一下棋盤上的棋子。三十餘名中階契師瞬間昏厥，不是被打中，是靈力被強制剝奪——那種剝奪不是破壞，而是「收回」，就像房東收回了租出去的東西。

有人大叫起來。有人試圖後退，有人試圖攻擊。

混亂在後排蔓延。

白無極沒動。

他站在那裡，看著那個老人，看著那些從地上翻卷起來的石塊在他身周繞了一圈，沒有一塊能真正砸到他——不是因為他躲開了，是因為那些石塊在接近他的瞬間就失去了「能傷害他」的可能性，Requiem讓一切傷害在抵達之前就消散於因果之中。

他清了清嗓子。

回頭看了一眼身後那些跪地、昏倒、茫然的人群。

然後重新看向契神。

契神也看著他。

「你。」老人開口，聲音依然平靜，「靠過來一些。」

白無極沒有靠過去，但也沒有退。他雙腿微微分開，擺出一個八極拳的起手式。

這個姿勢放在普通人身上看起來相當正常，放在此刻的天淵谷，放在這個全場最強者都被壓得抬不起頭的絕境之中，卻產生了某種奇異的效果——像是整個戰場的荒謬被他這一個姿勢給具象化了。

蘇念卿盯著他的背影，不知道為什麼，那股壓迫感輕了一些。

不是因為白無極做了什麼，只是因為他站在那裡。

有些人的存在本身就是定心丸，不在於他有多強，而在於他讓你知道：哪怕再荒謬的絕境，也有人選擇站著面對。

契神低下頭，那副普通老人的面孔上，浮現了一個很淡的表情。

很難定義那個表情，不是笑，不是怒，不是輕蔑，更不是讚賞。

只是某種久違的東西。

「千年了，」他再次輕聲說，這次似乎是對著白無極說的，「終於又有人讓我覺得有點意思了。」

他伸了個懶腰。

方圓五十里，天地法則同時震動，雲層翻湧，大地低鳴，遠處山峰的頂端無聲崩碎。

這是他「活動筋骨」的力道。

他還沒有出手。

白無極皺著眉頭，盯著對面那個老人，心裡轉過一個念頭：

這傢伙打起來應該很麻煩。

然後他想到了另一個問題，回頭對蘇念卿喊了一句：「你沒事吧？」

蘇念卿：「……你認真的嗎。」

「認真的，我看你臉色不太好。」

「我臉色一直這樣。」

「哦，那沒事了。」他重新轉向契神，「好，我們繼續。」

契神沉默片刻，說：「你很奇怪。」

「聽過。」白無極說，「不是第一個說這個的。」

後方，三個契聖對視了一眼。

他們的靈力都在硬撐，承受著那股無形的壓力，卻都沒有說話。

只是看著那個站在最前面、用最沒有高手架勢的姿勢、跟那個神說話的普通年輕人。

蘇長淵低聲說了一個字：「撐。」

撐住陣型，配合那個年輕人。

不是因為他們有把握，而是因為此刻他是唯一一個還能正常站著的人。

那就足夠了。
