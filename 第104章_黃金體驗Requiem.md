# 第104章　黃金體驗：Requiem

玄冥的爆發比白無極預計的還要猛烈。

第二次時停結束後，白無極在玄冥周圍又創造了一批生物，強化了原本還活著的那批，以為可以繼續消耗戰。但玄冥顯然也已經判斷出，再這樣耗下去對他沒有好處。

他選擇了硬摧毀。

不是針對那些生物，是針對白無極。

那道靈力爆發時，整個廢墟戰場的地面都顫動了一下，像是有什麼東西在地底深處猛烈跳動了一拍，然後靈力以絕對碾壓的方式向外傾洩——不是術式，不是有目標的攻擊，就是純粹的靈力洪流，覆蓋性的，大範圍的，讓所有在這個範圍內的東西，無論是生物還是人，全部承受這股壓力。

白無極周圍那些剩下的生物在那一瞬間全部失去靈識，碎裂，歸零。

那道靈力繼續前推，衝向白無極。

他沒有來得及再啟動時停——冷卻還沒有轉好，差了一口氣，那個感覺非常清晰，就差那麼一點點，而那一點點在現實戰鬥裡是一道不可逾越的牆。

他試圖用黃金體驗擋，黃金體驗的小人張開手，金色光芒向外延伸，轉化在靈力前端形成一個屏障——

玄冥的靈力洪流把那個屏障在零點三秒內撕開，然後打在白無極身上。

衝擊不是從一個方向來的，那個大範圍的洪流把他整個人卷起來，在廢墟裡滾了一段，撞在一面殘牆上，那面牆轟然崩塌，他被壓在底下。

靜了三秒。

碎石在他身上慢慢位移，他把自己從廢墟裡扒出來，一隻手撐著地面，試圖站起來。

傷了。

不是輕傷，他能感覺到胸口有幾根骨骼的狀態不對，呼吸的時候有一種撕拉感，是骨骼碎片在挫動軟組織。他這個世界的肉身沒有煉體，靠的是精神力護體，現在精神力大量消耗，護體層早就薄了，那一下吃到了玄冥全力洪流的邊緣，邊緣都這樣，正面的話……

白無極沒有往這個方向繼續想，因為沒有意義。

他把手支在一塊完整的石頭上，半跪著，呼吸調整了幾下，試圖把胸口那個感覺壓下去。

腦海裡，黃金體驗的小人從廢墟裡走出來，站在他面前，看著他。

它的臉上有一種白無極從未見過的表情。

那個表情，白無極盯著看了一秒，然後他懂了——那不是擔憂，那不是憤怒，那是一種非常深沉的、幾乎可以稱為「悲憫」的東西。

黃金體驗在悲憫他。

不是因為他受傷了，而是因為它看到了他承受的一切，它看到了他在這個世界裡為了什麼而戰，它看到了他在蘇念卿被扣押的瞬間那個「不行」背後的東西。

它一直都看得見，從一開始就看得見。

白無極在看到那個表情的瞬間，某種東西在他喉嚨裡哽了一下，不是難受，而是一種……被看見了的感覺。

他和這個金色小人相處了這麼久，它一直都在，陪他打每一場仗，陪他從一個什麼都不懂的穿越者慢慢摸索出這個世界的規則，陪他在深夜一個人坐著想「我他媽的為什麼在這裡」的時候也不聲不響地待在他身邊。

他從來沒有謝過它。

他也沒有問過它，這一切，對它來說是什麼感受。

「你，」白無極低聲說，喉嚨有點哽，「一直都知道，是嗎。」

黃金體驗沒有回答，它不會說話，但它伸出手，把手放在白無極垂著的那隻手的手背上。

那個接觸沒有任何溫度，沒有任何重量，他感覺不到任何物理意義上的觸碰——但他感覺到了。

玄冥的腳步聲從廢墟那邊傳來，穩定而緩慢，那是一個有絕對把握的人走路的聲音。

白無極深吸了一口氣，那口氣帶著胸口的撕拉感，他忍著，把氣息調平，試圖站起來。

腿有點軟，他強迫自己站直了。

「你已經到了極限，」玄冥的聲音從廢墟後面傳過來，平靜，「那個奇異的精神力是你的上限，但你的身體不是這個世界的靈契者，它很脆弱。你沒有更多牌了。」

白無極扶著殘牆站著，低著頭，頭髮有幾縷垂下來擋住了他的眼睛。

玄冥繞過廢墟，出現在他的視野裡，距離很近，他走的這段路不緊不慢，顯然不認為這個年輕人還有什麼反擊能力。

「你做得很好，」玄冥說，在他這個位置這是一種很高的評價，「你讓我付出了比預期更多的代價。但結果不會改變。」

他舉起右手。

靈力再次凝聚，這一次不是大範圍的洪流，是針對一個目標的、精準的、足以穿透現在這個年輕人的殘餘防禦的精準打擊。

白無極看著那道靈力成形。

然後他感覺到了一件事——腦海裡，那顆金球，不是白金之星，是黃金體驗，那顆金球，在他重傷之後，反而以一種奇異的方式，達到了它從來沒有達到過的一個頻率。

那個頻率，讓他想起一個詞。

極限。

黃金體驗的小人抬起頭，看著他。那個悲憫的表情已經不在了，換成的東西更深沉，更安靜，一種白無極用語言很難準確描述的東西——像是「完成了」，但又不是那麼簡單。

玄冥的攻擊打出來了。

那道靈力穿過廢墟的空氣，向白無極飛來，速度極快，那種精準性代表它會完整地打在白無極胸口——

打在了他身上。

然後消失了。

不是被擋住，不是被偏轉，是消失了，像是一道筆畫被橡皮擦擦過，像是這件事從來沒有發生過，那道靈力不在了，白無極身上沒有任何新的傷，廢墟的地面也沒有任何多餘的痕跡。

玄冥站著，怔了一秒。

然後他再打。

同樣的消失。

再打。

消失。

三次，每一次一樣的消失，像是這個世界在某個根本性的層面上，已經決定了「這件事不會發生」，所以不管玄冥多少次啟動攻擊，多少次讓靈力飛向那個目標，因果在最源頭的地方就已經把那個結果抹除了。

玄冥停下來，第一次在這場戰鬥裡，真正無法判斷眼前發生的事情。

白無極低著頭，也怔了一秒。

他感覺著自己的身體——第三下之後，他確定了，攻擊確實打在他身上了，他能感覺到那個衝擊落下來的那一刻，然後那個衝擊消失了，連帶著那一刻發生的一切，一起消失了。

不是護身，不是防禦，而是——那件事，從因果上，就沒有發生過。

白無極緩緩抬起頭。

他看著黃金體驗的小人，那個小人也看著他，那個「完成了」的表情，白無極現在能更清楚地讀懂它了。

那個表情說的是：這是我最後能給你的東西。

「這是什麼？」白無極低聲問，不是問任何人，是問那個現在在他精神海裡以全新姿態存在的東西。

黃金體驗。

Requiem。

因果歸零。

任何對他的攻擊，任何試圖傷害他的行為，都永遠無法到達真實的他——不是打不穿，而是在因果的最源頭，那個「打到他」的結果就不存在。

「……」

白無極站著，把這個能力的意義在腦子裡理了一圈，理出來之後，他站著沉默了大概五秒。

然後玄冥第四次抬手。

打在他身上。

消失。

玄冥：「……」

白無極：「……」

戰場上的風吹過，帶走一些碎石塵土，也帶走一些奇怪的沉默。

「所以，」白無極慢慢開口，聲音有點奇怪，像是一個試圖用平靜語氣描述一件完全不平靜的事的人，「我現在是……打不到的那種？」

玄冥沒有回答，他的眼神不再是評估，是真正的迷惘。

白無極低頭看了一眼黃金體驗，它站在他旁邊，穩穩的，那身金色光芒比以往任何時候都更深，更實在，像是一個終於到達了自己應該在的地方的東西。

因果歸零。

Requiem。

白無極緩緩吐出一口氣。

然後他感覺了一下Eternal的冷卻，已經轉好了。

他直起腰，把那幾根碎肋骨的不對勁壓到意識底層，腿站直，面對玄冥，臉上的表情從那種沉默回到了一種更平靜的東西。

「好，」他說，「現在換我問你了。」

玄冥沒有開口。

「你打算怎麼辦？」
