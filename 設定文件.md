# 《替身無極》設定文件

---

## 一、主角

### 白無極

- **穿越前**：普通上班族，業餘練八極拳多年，實戰派
- **性格**：仗助型——平時隨和搞笑、嘴砲擔當，碰到底線就爆發，反差帥
- **外號/稱呼**：朋友叫「老白」，敵人聽到「白無極」三個字就頭皮發麻
- **穿越原因**：腦海中先天存在一顆精神力金球，某天金球共鳴引發穿越，抵達異世界後金球孵化，誕生兩個替身

### 戰鬥風格（三層次）

| 層次 | 手段 | 使用時機 |
|------|------|----------|
| 日常 | 八極拳（崩拳、鐵山靠、貼身短打） | 一般對手，不想暴露底牌 |
| 認真 | 白金之星 / 黃金體驗 單體出擊 | 遇到強敵 |
| 全力 | 雙替身齊出，一體三身 | 生死之戰 |

## 二、替身系統

### 來源

精神力金球孵化後誕生，殘留大量未開發潛力。每次經歷極限戰鬥、精神突破、或強烈情感衝擊，金球再次共鳴，解鎖替身新層次。

### 核心規則

- 替身只有替身使者能看見，異世界的人完全看不到
- 靈契者能感應到「有力量存在」但無法判斷形態和來源
- 替身無代價，在這個「力量必有代價」的世界中屬於規則之外的存在
- 兩個替身可同時出現（一體三身）

### 白金之星（Star Platinum）進化路線

| 階段 | 名稱 | 能力 | 觸發條件 |
|------|------|------|----------|
| 初始 | 白金之星 | 近戰格鬥、超精密動作、超高速 | 金球孵化 |
| 二階 | 白金之星：Over Drive | 時間停止（1秒） | 第一次生死危機 |
| 三階 | 白金之星：The World | 時停延長至5秒，時停中可移動物體 | 重大戰役突破 |
| 四階 | 白金之星：Eternal | 時停10秒，可在時停中使用黃金體驗 | 精神極限突破 |
| 終階 | 白金之星：Over Heaven | 時間相關究極能力（改寫因果） | 最終覺醒 |

### 黃金體驗（Gold Experience）進化路線

| 階段 | 名稱 | 能力 | 觸發條件 |
|------|------|------|----------|
| 初始 | 黃金體驗 | 無機物→生物、基礎治療 | 金球孵化 |
| 二階 | 黃金體驗：Life | 創造的生物可持續存在且可操控 | 對生命有新的理解 |
| 三階 | 黃金體驗：Genesis | 創造大型生物、改造地形生態 | 大規模戰役需求 |
| 四階 | 黃金體驗：Requiem | 因果歸零——任何對主角的攻擊永遠無法到達真實 | 極限覺醒 |
| 終階 | 黃金體驗：Act of God | 創造生命＋賦予生命替身能力（造兵器） | 最終覺醒 |

### 雙替身融合型態（隱藏線）

> **白金體驗：Star of Creation**
> 兩個替身同時進化至四階以上觸發。
> 時間停止 + 生命創造同時發動 = 在凍結的時間中創造無限生命。
> 契神級都無法理解的力量，也是白無極被召喚到這個世界的真正原因。

### 白金之星 vs 黃金體驗互補表

| 弱點 | 白金之星的問題 | 黃金體驗來補 |
|------|----------------|--------------|
| 射程 | 近距離只有2米 | 創造生物遠程偵查、攻擊 |
| 治療 | 不能治自己 | 黃金體驗造器官修自己 |
| 偵查 | 零情報能力 | 創造小動物當眼線 |
| 戰術 | 只會用拳頭 | 把環境變成生物製造陷阱 |

| 弱點 | 黃金體驗的問題 | 白金之星來補 |
|------|----------------|--------------|
| 破壞力 | C級打不動 | 白金之星A級碾壓 |
| 正面戰鬥 | 扛不住近戰 | 白金之星貼臉無敵 |
| 速度壓制 | 快攻型沒轍 | 時間停止完全解決 |

## 三、異世界——天衍大陸

### 核心力量：靈契

人與天地萬物簽訂契約，借用自然之力。契約越強，代價越大。
**世界常識：力量必有代價。**

### 靈契等級

| 等級 | 稱號 | 能力範圍 | 代價 |
|------|------|----------|------|
| 一階 | 契徒 | 借用單一元素微弱之力（點火、引水） | 體力消耗 |
| 二階 | 契士 | 元素成形可攻擊（火球、冰槍） | 精神疲勞 |
| 三階 | 契師 | 與靈獸締結契約，人獸合擊 | 壽命（每次減壽數月） |
| 四階 | 契將 | 領域級力量，可覆蓋一片區域 | 壽命（每次減壽數年）+ 記憶模糊 |
| 五階 | 契王 | 與天地法則締約（重力、時間碎片等） | 獻祭情感（逐漸失去喜怒哀樂） |
| 六階 | 契聖 | 傳說級，能改寫局部天地規則 | 獻祭自我（最終忘記自己是誰） |
| 七階 | 契神 | 只存在於傳說中 | ？？？ |

### 靈契三大分支

| 分支 | 特點 | 代表 |
|------|------|------|
| 元素契 | 與火、水、風、雷、土等自然元素締約 | 最常見，軍隊主力 |
| 獸契 | 與靈獸締約，召喚靈獸作戰 | 靈獸越強代價越大 |
| 體契 | 將契約刻入自身肉體，強化體質 | 最少見，近戰最強 |

### 替身等級 vs 靈契等級對照

| 替身階段 | 該等級無敵 | 能越級打贏 | 有壓力 |
|----------|------------|------------|--------|
| 初始態 | 契師無敵 | 能打贏契將 | 契王以上 |
| 二階 | 契將無敵 | 能打贏契王 | 契聖 |
| 三階 | 契王無敵 | 能打贏契聖 | 契神 |
| 四階 | 契聖無敵 | 能挑戰契神 | — |
| 終階/融合 | 契神無敵 | 天下無敵 | — |

### 替身的天然優勢

- 看不見 → 靈契者永遠在跟「未知」戰鬥
- 時停無法被感知 → 不知道自己怎麼輸的
- 無代價 → 靈契者打久了要考慮壽命消耗，白無極可以無限打
- 測不出等級 → 本身就是最大的裝逼

## 四、女主角

### 蘇念卿

- **身份**：天衍大陸四大家族之一「蘇家」的天才嫡女
- **靈契**：獸契師，契約靈獸為「霜翎」——一隻冰屬性銀色靈狐，外形優雅但戰力極強
- **等級**：出場時契師巔峰，年紀輕輕已是天才級
- **性格**：外冷內熱，面癱毒舌，但對認可的人非常護短。跟白無極的相處模式是互懟互嘴但默契十足
- **與主角的關係**：唯一女主，不搞後宮
  - 初見：白無極用八極拳打贏她的靈獸，她不服氣開始跟蹤研究他
  - 中期：並肩作戰，她是第一個「隱約感覺到替身存在」的人
  - 後期：她能感知替身的存在成為關鍵劇情線（是否與她的獸契天賦有關？）
- **戰鬥風格**：靈狐「霜翎」負責遠程冰系攻擊 + 主人近身冰甲防禦，攻守兼備
- **成長線**：從「蘇家天才」到「跳脫家族束縛走自己的路」，白無極是她的催化劑

### 白無極 × 蘇念卿 互動風格

| 場景 | 互動 |
|------|------|
| 日常 | 互嘴鬥嘴，老白耍寶她翻白眼 |
| 戰鬥 | 默契配合，霜翎冰封 + 白金之星收割 |
| 認真時刻 | 少數她露出真心的瞬間，反差萌 |

## 五、故事節奏

### 爽感循環

```
每個大階段：
1. 碾壓當前級別 → 觀眾震驚、打臉
2. 越級挑戰 → 苦戰但贏，名聲大噪
3. 遇到天花板 → 被壓制
4. 觸發進化 → 反殺翻盤
5. 新的碾壓循環開始
```

### 裝逼節奏

```
異世界的人測不出白無極的等級
→ 以為他是體契者（因為八極拳）
→ 替身一出全場懵逼「什麼東西？！看不見！」
→ 每次暴露新能力都是一次認知崩塌
→ 永遠在被低估，永遠在打臉
```

## 六、寫作原則

- **純爽文**，爽就完事了
- **不搞後宮**，一男一女感情線清晰
- **主角無代價限制**，白金之星 + 黃金體驗完美互補
- **戰鬥層次分明**：八極拳 → 單替身 → 雙替身 → 融合態，底牌一層層翻
- **節奏明快**，不拖泥帶水
